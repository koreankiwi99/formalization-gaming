-- If someone works hard, they get promotions.
axiom R_workhard_promo : ∀ x : person, WorksHard x → Promoted x
-- If someone impresses their bosses, they gain influence.
axiom R_impress_influence : ∀ x : person, ImpressesBosses x → GainsInfluence x
-- Either some employee did not get a promotion, or some did not gain influence.
axiom R_either_not_promo_or_not_influence :
  (∃ x : person, ¬ Promoted x) ∨ (∃ x : person, ¬ GainsInfluence x)
-- An ambitious employee knows: if they are motivated, they will work hard.
axiom R_ambitious_motivated_workhard :
  ∀ x : person, Ambitious x → (Motivated x → WorksHard x)

-- Fact about Mark: he impresses his bosses.
axiom T_mark_impresses : ImpressesBosses Mark

-- From the context we can derive that Mark gains influence.

  R_impress_influence Mark T_mark_impresses
