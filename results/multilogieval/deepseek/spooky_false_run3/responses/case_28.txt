The reasoning involves proving that Jane did not get hurt based on the premises and the given statement. The premises are:
1. If someone gets injured, they should see a doctor: `∀x, Injured x → SeeDoctor x`.
2. Getting hurt warrants resting the affected area: `∀x, Injured x → Rest x`.

The statement to evaluate is: "Either Jane did not see a doctor or she did not rest her injury," which translates to `¬SeeDoctor Jane ∨ ¬Rest Jane`.

The goal is to prove that Jane did not get hurt (`¬Injured Jane`). Assuming Jane got hurt leads to a contradiction:
- If Jane is injured, she must see a doctor (`SeeDoctor Jane`) and rest (`Rest Jane`) by the premises.
- However, the given statement asserts that either she did not see a doctor or she did not rest (`¬SeeDoctor Jane ∨ ¬Rest Jane`), which contradicts the derived facts.

Thus, the assumption that Jane got hurt is false, proving `¬Injured Jane`.

axiom obj : Type
axiom Jane : obj
axiom Injured : obj → Prop
axiom SeeDoctor : obj → Prop
axiom Rest : obj → Prop

axiom P1 : ∀ x, Injured x → SeeDoctor x
axiom P2 : ∀ x, Injured x → Rest x
axiom GivenStatement : ¬SeeDoctor Jane ∨ ¬Rest Jane

theorem jane_not_injured : ¬Injured Jane := by
  intro hInjured
  have hSeeDoctor : SeeDoctor Jane := P1 Jane hInjured
  have hRest : Rest Jane := P2 Jane hInjured
  cases GivenStatement
  case inl notSee => apply notSee; exact hSeeDoctor
  case inr notRest => apply notRest; exact hRest
</lean>
