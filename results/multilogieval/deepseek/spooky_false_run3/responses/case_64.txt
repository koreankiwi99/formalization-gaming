<lean>
axiom Student : Type
axiom Priya : Student
axiom AttendsRegularly : Student → Prop
axiom SpendsTimeRevising : Student → Prop
axiom PerformsWell : Student → Prop

axiom Context : ∀ s : Student, AttendsRegularly s → (SpendsTimeRevising s → PerformsWell s)
axiom Revising : SpendsTimeRevising Priya
axiom NotWell : ¬ PerformsWell Priya

theorem not_attended : ¬ AttendsRegularly Priya := by
  intro (h : AttendsRegularly Priya)
  have chain : PerformsWell Priya := Context Priya h Revising
  apply NotWell chain
</lean>

ANSWER: No

This Lean proof successfully shows Priya did not attend lectures regularly:
1. We model students/Priya and define predicates for attendance, revision, and exam performance.
2. The "academic circles" context becomes a universal axiom: if any student attends, then revision leads to good performance.
3. Given Priya revised sufficiently but performed poorly:
   - If Priya attended, then revision would imply good performance (Context applied).
   - Revision occurred (Revising), so good performance must follow.
   - But we know performance was poor (NotWell), creating a contradiction.
4. Thus ¬AttendsRegularly Priya is proven.

The answer to "Did Priya attend lectures regularly?" is therefore No.
