1. If John went to bed late (assumption), he got up late (P5).
2. If he got up late, he missed the bus (P4).
3. If he missed the bus, he was late for school (P2).
4. John knows ¬time_to_get_ready (P6), and since alarm_went_off → time_to_get_ready (P1), we derive ¬alarm_went_off (via contraposition).
5. Premise P3: alarm_went_off ∨ ¬late_for_school. 
   - If alarm_went_off, it contradicts ¬alarm_went_off (step 4).
   - If ¬late_for_school, it contradicts being late for school (step 3).
Both cases of P3 lead to contradiction when combined with steps 3 and 4.

axiom time_to_get_ready : Prop
axiom miss_bus : Prop
axiom late_for_school : Prop
axiom get_up_late : Prop
axiom go_to_bed_late : Prop

axiom P1 : alarm_went_off → time_to_get_ready
axiom P2 : miss_bus → late_for_school
axiom P3 : alarm_went_off ∨ ¬ late_for_school
axiom P4 : get_up_late → miss_bus
