<lean>
axiom obj : Type
axiom Mary : obj
axiom Problems : obj → Prop
axiom Restart : obj → Prop
axiom InstallUpdates : obj → Prop

-- Premises from context
axiom A1 : ∀ x : obj, Problems x → Restart x
axiom A2 : ∀ x : obj, Problems x → InstallUpdates x

-- Given statement about Mary
axiom Fact : ¬Restart Mary ∨ ¬InstallUpdates Mary

-- Prove Mary did NOT have computer problems
theorem mary_no_problems : ¬Problems Mary := 
  fun h : Problems Mary => 
  have restart_Mary : Restart Mary := A1 Mary h
  have updates_Mary : InstallUpdates Mary := A2 Mary h
  show False from 
    match Fact with
    | Or.inl not_restart => not_restart restart_Mary
    | Or.inr not_updates => not_updates updates_Mary
</lean>

ANSWER: No

The proof demonstrates that assuming Mary had computer problems leads to a contradiction. From the premises:
1. If Mary had problems, she would restart (A1)
2. If Mary had problems, she would install updates (A2)
But the given fact states she either didn't restart or didn't install updates. Both cases contradict the derived restart/update actions, proving ¬Problems(Mary).
