The context states that the weather was sunny ("It was a sunny day") and provides several implications. However, when assuming Alex got sunburnt, we derive a contradiction because the sunny weather leads to Alex not being sunburnt. Specifically, sunny weather implies warm water, which causes Alex to go swimming. Swimming gives exercise, and since Alex exercised, they must have applied sunscreen (from the disjunction). Applying sunscreen means Alex is not sunburnt, contradicting the assumption. Thus, we prove that if Alex got sunburnt, the weather was **not** sunny, answering the embedded question.

<lean>
axiom obj : Type
axiom Alex : obj
axiom Sunny : Prop
axiom WarmWater : Prop
axiom Swimming : obj → Prop
axiom Exercise : obj → Prop
axiom AppliedSunscreen : obj → Prop
axiom Sunburnt : obj → Prop

axiom T1 : Sunny
axiom R1 : AppliedSunscreen Alex → ¬ Sunburnt Alex
axiom R2 : ∀ (p : obj), Swimming p → Exercise p
axiom R3 : AppliedSunscreen Alex ∨ ¬ Exercise Alex
axiom R4 : WarmWater → Swimming Alex
axiom R5 : Sunny → WarmWater

axiom H : Sunburnt Alex

theorem not_sunny : ¬ Sunny := 
  by
    intro s
    have ww : WarmWater := R5 s
    have swim : Swimming Alex := R4 ww
    have exercise : Exercise Alex := R2 Alex swim
    cases R3 with
    | inl a =>
        have no_burn := R1 a
        exact no_burn H
    | inr no_ex =>
        exact no_ex exercise
</lean>
