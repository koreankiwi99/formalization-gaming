The reasoning involves proving that the wood was not of high quality based on the given premises. The general rule states that high-quality wood and precise joints imply a sturdy product. However, the joints were precise but the bookshelf was not sturdy. Therefore, the wood cannot be high quality.

<lean>
axiom obj : Type
axiom wood : obj
axiom joints : obj
axiom bookshelf : obj
axiom HighQuality : obj → Prop
axiom CutPrecisely : obj → Prop
axiom SturdyAndLongLasting : obj → Prop
axiom R1 : HighQuality wood → (CutPrecisely joints → SturdyAndLongLasting bookshelf)
axiom T1 : CutPrecisely joints
axiom T2 : ¬SturdyAndLongLasting bookshelf
theorem not_high_quality : ¬HighQuality wood :=
  fun h : HighQuality wood =>
    have h1 : CutPrecisely joints → SturdyAndLongLasting bookshelf := R1 h
    have h2 : SturdyAndLongLasting bookshelf := h1 T1
    T2 h2
</lean>
