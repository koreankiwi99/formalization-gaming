```lean
<lean>
axiom obj : Type
axiom Simone : obj
axiom TrainDedication : obj → Prop
axiom AdequateRest : obj → Prop
axiom PerformPeak : obj → Prop

-- Context: If someone trains with dedication, then if they get adequate rest, they will perform at peak
axiom coach_saying : ∀ x : obj, TrainDedication x → (AdequateRest x → PerformPeak x)

-- Facts: Simone got adequate rest but did not perform at peak
axiom simone_rest : AdequateRest Simone
axiom simone_not_peak : ¬ PerformPeak Simone

-- Prove Simone did NOT train with dedication
theorem simone_not_dedicated : ¬ TrainDedication Simone :=
  fun h : TrainDedication Simone => 
    have h1 : AdequateRest Simone → PerformPeak Simone := coach_saying Simone h
    have h2 : PerformPeak Simone := h1 simone_rest
    simone_not_peak h2
</lean>
```

ANSWER: No
