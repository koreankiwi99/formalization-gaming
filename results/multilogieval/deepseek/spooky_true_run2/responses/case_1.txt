The premises state:
1. If the team plays strong perimeter defense, they force many turnovers.
2. If the big men protect the paint, they limit inside scoring.
3. For the Lakers specifically: either the perimeter defense is strong or the paint is protected (or both).

The question asks: If the Lakers did not limit inside scoring, did they force many turnovers?

Proof strategy:
- Assume the Lakers did not limit inside scoring.
- From premise 3, the Lakers either have strong perimeter defense or protected the paint (or both).
  - Case 1: Strong perimeter defense. Then by premise 1, they force many turnovers.
  - Case 2: Protected the paint. Then by premise 2, they limit inside scoring. But this contradicts our assumption that they did not limit inside scoring. Thus, this case is impossible, and Case 1 must hold.

Therefore, if the Lakers did not limit inside scoring, they must have forced many turnovers.

<lean>
axiom obj : Type
axiom Lakers : obj
axiom PerimeterStrong : obj → Prop
axiom ForceTurnovers : obj → Prop
axiom ProtectPaint : obj → Prop
axiom LimitInside : obj → Prop

axiom perimeter_to_turnovers : ∀ (t : obj), PerimeterStrong t → ForceTurnovers t
axiom paint_to_inside : ∀ (t : obj), ProtectPaint t → LimitInside t
axiom lakers_disjunction : PerimeterStrong Lakers ∨ ProtectPaint Lakers

theorem lakers_result : ¬ LimitInside Lakers → ForceTurnovers Lakers := by
  intro h
  cases lakers_disjunction
  case inl ps => exact perimeter_to_turnovers Lakers ps
  case inr pp => 
        have lim := paint_to_inside Lakers pp
        contradiction
</lean>

ANSWER: Yes
