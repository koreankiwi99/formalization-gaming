The reasoning involves analyzing the given premises and the statement about last quarter's sales. The key steps are:

1. From the context:
   - Investing in marketing leads to more customers, which increases sales: `InvestMarketing → SalesIncrease`
   - The budget allocation: `InvestMarketing ∨ ProductionCostsIncreased`

2. Given that sales did not increase (`¬SalesIncrease`), we apply modus tollens to the first implication:
   - `¬SalesIncrease → ¬InvestMarketing`

3. With `¬InvestMarketing` established, we examine the disjunction:
   - Since `InvestMarketing` is false, `ProductionCostsIncreased` must be true to satisfy `InvestMarketing ∨ ProductionCostsIncreased`

Thus, production costs increased is proven.

<lean>
axiom InvestMarketing : Prop
axiom SalesIncrease : Prop
axiom ProductionCostsIncreased : Prop

axiom marketing_chain : InvestMarketing → SalesIncrease
axiom budget_disjunction : InvestMarketing ∨ ProductionCostsIncreased
axiom given_no_increase : ¬ SalesIncrease

theorem production_costs_increased : ProductionCostsIncreased :=
  Or.elim budget_disjunction
    (fun h_invest : InvestMarketing =>
      have : SalesIncrease := marketing_chain h_invest
      absurd this given_no_increase)
    (fun h_pc : ProductionCostsIncreased => h_pc)

</lean>

ANSWER: Yes
