The context provides several premises about the office scenario, and the question asks whether John is diligent if his coworkers are not hungry. The goal is to prove that the answer is "No" by showing that if John's coworkers are not hungry, then John is not diligent.

### Formalization:
- `L`: It is close to lunchtime.
- `H`: John's coworkers are hungry.
- `P`: John prepares well for the presentation.
- `R`: John is ready for the meeting.
- `D`: John is diligent.

### Axioms:
1. If it is close to lunchtime, then coworkers are hungry: `L → H`.
2. If John prepares well, then he is ready: `P → R`.
3. Either it is close to lunchtime, or John is not ready, or both (simplified to `L ∨ ¬R`).
4. If John is diligent, then he prepares well: `D → P`.

### Proof Strategy:
To prove "No" (i.e., if coworkers are not hungry, then John is not diligent), we show `¬H → ¬D`:
1. Assume `¬H` (coworkers are not hungry).
2. From `L → H` and `¬H`, derive `¬L` (not close to lunchtime).
3. From `L ∨ ¬R` and `¬L`, derive `¬R` (John is not ready).
4. From `P → R` and `¬R`, derive `¬P` (John did not prepare well).
5. From `D → P` and `¬P`, derive `¬D` (John is not diligent).

This proves `¬H → ¬D`, meaning if coworkers are not hungry, John is not diligent. Thus, the answer to the question is "No".

axiom A1 : L → H
axiom A2 : P → R
axiom A3 : L ∨ ¬R
axiom A4 : D → P

theorem not_hungry_implies_not_diligent : ¬H → ¬D :=

  have not_L : ¬L := fun l : L => h (A1 l)
