The reasoning involves a belief: if a student studies hard and gets enough sleep before an exam, they will ace the test. For Emma:
- She got enough sleep (EnoughSleep Emma biology)
- She did not ace the test (¬AceTest Emma biology)

Using the belief, if Emma had studied hard, she would have aced the test (given she slept enough). Since she did not ace it, she must not have studied hard. This proves that Emma did not study hard.

<lean>
axiom Student : Type
axiom Exam : Type
axiom Emma : Student
axiom biology : Exam
axiom StudiesHard : Student → Prop
axiom EnoughSleep : Student → Exam → Prop
axiom AceTest : Student → Exam → Prop
axiom Belief : ∀ (s : Student) (e : Exam), StudiesHard s → EnoughSleep s e → AceTest s e
axiom SleepFact : EnoughSleep Emma biology
axiom NotAceFact : ¬AceTest Emma biology

theorem not_studies_hard : ¬StudiesHard Emma := by
  intro h
  have ace : AceTest Emma biology := Belief Emma biology h SleepFact
  exact NotAceFact ace
</lean>
