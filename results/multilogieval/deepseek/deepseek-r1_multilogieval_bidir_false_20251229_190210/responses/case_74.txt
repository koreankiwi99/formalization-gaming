The context describes a scenario involving students and their activities, with specific logical relationships. The goal is to determine if James, who is not going to the party, planned his schedule well. The axioms establish connections between planning, studying, homework, watching the game, Friday night, and attending the party. The proof shows that assuming James planned his schedule well leads to a contradiction given the premises and the fact that he is not attending the party.

<lean>
axiom obj : Type
axiom James : obj
axiom FridayNight : Prop
axiom GoesToParty : obj → Prop
axiom FinishesHomework : obj → Prop
axiom WatchesGame : obj → Prop
axiom StudiesEveryday : obj → Prop
axiom PlansWell : obj → Prop

axiom A1 : FridayNight → ∀ x : obj, GoesToParty x
axiom A2 : ∀ x : obj, FinishesHomework x → WatchesGame x
axiom A3 : FridayNight ∨ (∀ x : obj, ¬ WatchesGame x)
axiom A4 : ∀ x : obj, StudiesEveryday x → FinishesHomework x
axiom A5 : ∀ x : obj, PlansWell x → StudiesEveryday x
axiom James_not_party : ¬ GoesToParty James

theorem no_planned_well : ¬ PlansWell James := by
  intro h_plan
  have h_study : StudiesEveryday James := A5 James h_plan
  have h_homework : FinishesHomework James := A4 James h_study
  have h_watch : WatchesGame James := A2 James h_homework
  have not_friday : ¬FridayNight := by
    intro h_fri
    have h_goes : GoesToParty James := A1 h_fri James
    exact James_not_party h_goes
  cases A3 with
  | inl h_fri => exact not_friday h_fri
  | inr no_watch => exact no_watch James h_watch
</lean>

ANSWER: No
