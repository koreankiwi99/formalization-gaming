The context states that if a company invests in marketing (MarketingFocus), sales increase (SalesIncrease). Additionally, either MarketingFocus, ProductionCostIncreased, or both occurred. Given that sales did not increase (¬SalesIncrease), we can deduce that MarketingFocus must be false (because if MarketingFocus were true, SalesIncrease would be true, contradicting ¬SalesIncrease). Since MarketingFocus is false, the disjunction (MarketingFocus ∨ ProductionCostIncreased) forces ProductionCostIncreased to be true.

<lean>
axiom MarketingFocus : Prop
axiom ProductionCostIncreased : Prop
axiom SalesIncrease : Prop

axiom invest_in_marketing : MarketingFocus → SalesIncrease
axiom factors : MarketingFocus ∨ ProductionCostIncreased

theorem cost_increased : ¬SalesIncrease → ProductionCostIncreased := by
  intro hNoSales
  have hNoMarket : ¬MarketingFocus := by
    intro hMarket
    apply hNoSales
    exact invest_in_marketing hMarket
  cases factors with
  | inl hMarket => exact False.elim (hNoMarket hMarket)
  | inr hCost => exact hCost
</lean>

ANSWER: Yes
