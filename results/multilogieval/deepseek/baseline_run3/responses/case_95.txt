The context provides several implications and a disjunction. The key reasoning is as follows:

1. If Mary did not meet friends (¬MeetFriends), then by contrapositive of "errands done → meet friends", her errands were not done (¬ErrandsDone).
2. The disjunction "clear weather ∨ errands done" must hold. With ¬ErrandsDone, it follows that the weather cleared (Clear).
3. Clear weather implies the game goes on (GameOn), which leads to fans cheering (Cheer), resulting in an exciting night (Exciting).

Thus, ¬MeetFriends implies Exciting. The Lean proof formalizes this reasoning.

<lean>
axiom Clear : Prop
axiom ErrandsDone : Prop
axiom GameOn : Prop
axiom MeetFriends : Prop
axiom Cheer : Prop
axiom Exciting : Prop

axiom A1 : Clear → GameOn
axiom A2 : ErrandsDone → MeetFriends
axiom A3 : Clear ∨ ErrandsDone
axiom A4 : GameOn → Cheer
axiom A5 : Cheer → Exciting

theorem T : ¬MeetFriends → Exciting :=
  fun h : ¬MeetFriends =>
  have h1 : ¬ErrandsDone := fun h_errands => h (A2 h_errands)
  have h2 : Clear :=
    Or.elim A3
      (fun h_clear => h_clear)
      (fun h_errands => absurd h_errands h1)
  have h3 : GameOn := A1 h2
  have h4 : Cheer := A4 h3
  show Exciting from A5 h4
</lean>

ANSWER: Yes
