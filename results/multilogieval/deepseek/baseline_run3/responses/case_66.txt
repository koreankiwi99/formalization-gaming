The context states that if a company develops an innovative product and markets it effectively, then the product will be commercially successful. For Acme Inc., we know they marketed effectively but were not commercially successful. We need to determine if they developed an innovative product.

Using logical reasoning:
- Let `innovative` mean "Acme developed an innovative product".
- Let `marketed` mean "Acme marketed effectively".
- Let `successful` mean "The product was commercially successful".
- The context: `innovative → (marketed → successful)`
- Given: `marketed` and `¬successful`

From `innovative → (marketed → successful)` and `¬successful`, we derive that if `innovative` were true, then `marketed → successful` would hold. Since `marketed` is true, this would imply `successful`, contradicting `¬successful`. Thus, `innovative` must be false.

Lean formalization and proof:

<lean>
axiom innovative : Prop
axiom marketed : Prop
axiom successful : Prop
axiom R1 : innovative → (marketed → successful)
axiom T1 : marketed
axiom T2 : ¬ successful

theorem not_innovative : ¬ innovative := by
  intro h
  have h1 : marketed → successful := R1 h
  have h2 : successful := h1 T1
  exact T2 h2
</lean>

The Lean code proves `¬ innovative`, meaning Acme did not develop an innovative product. Therefore, the answer to the question "Did Acme Inc. develop an innovative product?" is **No**.

ANSWER: No
