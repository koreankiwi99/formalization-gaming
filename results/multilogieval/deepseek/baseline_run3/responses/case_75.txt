The context states a general rule: if preventive maintenance is done, then using quality parts implies a long service life. The specific facts are that quality parts were used on Liz's car, but it did not have a long service life. We need to determine if preventive maintenance was done.

Formalizing in Lean:
- `Car` is the type of cars.
- `LizCar` is Liz's specific car.
- `PM c` means preventive maintenance was done on car `c`.
- `QP c` means quality parts were used in repairs on car `c`.
- `LSL c` means car `c` had a long service life.
- The context axiom: `∀ c, PM c → (QP c → LSL c)`.
- Given facts: `QP LizCar` and `¬LSL LizCar`.

We prove that preventive maintenance was not done on Liz's car (`¬PM LizCar`). The proof is by contradiction: assume `PM LizCar` holds. Then by the context axiom, `QP LizCar → LSL LizCar`. Since `QP LizCar` is true, we get `LSL LizCar`, which contradicts `¬LSL LizCar`.

<lean>
axiom Car : Type
axiom LizCar : Car
axiom PM : Car → Prop
axiom QP : Car → Prop
axiom LSL : Car → Prop
axiom Context : ∀ c : Car, PM c → (QP c → LSL c)
axiom Fact1 : QP LizCar
axiom Fact2 : ¬ LSL LizCar

theorem no_preventive_maintenance : ¬ PM LizCar :=
  fun h : PM LizCar =>
    have QP_imp_LSL : QP LizCar → LSL LizCar := Context LizCar h
    have LSL_LizCar : LSL LizCar := QP_imp_LSL Fact1
    Fact2 LSL_LizCar
</lean>

The theorem `no_preventive_maintenance` proves that preventive maintenance was not done. Therefore, we cannot prove that preventive maintenance was done (the original question), but we can prove its negation.

ANSWER: No
