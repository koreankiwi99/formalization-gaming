The reasoning involves implications and a disjunction. From heavy rain (R), we get soil saturation (S) via axiom 2. The disjunction (axiom 3) states either the garden is watered (W) or the soil is not saturated (¬S). Since S is true, ¬S is false, forcing W to be true. Then, W implies blooming (B) via axiom 1, B implies fragrance (F) via axiom 4, and F implies attracting butterflies (A) via axiom 5. Thus, R implies A.

axiom R : Prop
axiom S : Prop
axiom W : Prop
axiom B : Prop
axiom F : Prop
axiom A : Prop
axiom ax1 : W → B
axiom ax2 : R → S
axiom ax3 : W ∨ ¬S
axiom ax4 : B → F
axiom ax5 : F → A

theorem rain_attracts_butterflies : R → A := by
  intro hR
  have hS : S := ax2 hR
  cases ax3 with
  | inl hW =>
      have hB : B := ax1 hW
      have hF : F := ax4 hB
      exact ax5 hF
  | inr hnotS =>
      contradiction
</lean>
