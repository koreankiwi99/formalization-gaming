The context describes general conditions about learning guitar and entertaining friends. Emma is specified as unable to entertain friends. The reasoning shows that if Emma cannot entertain friends, she must have taken guitar lessons (since she didn't learn basic chords), and taking lessons implies she gains music theory knowledge.

<lean>
axiom Person : Type
axiom Emma : Person
axiom LearnedBasic : Person → Prop
axiom TookLessons : Person → Prop
axiom CanPlaySongs : Person → Prop
axiom CanEntertain : Person → Prop
axiom KnowsMusicTheory : Person → Prop

axiom A1 : ∀ p : Person, LearnedBasic p → CanPlaySongs p
axiom A2 : ∀ p : Person, CanPlaySongs p → CanEntertain p
axiom A3 : ∀ p : Person, LearnedBasic p ∨ TookLessons p
axiom A4 : ∀ p : Person, TookLessons p → KnowsMusicTheory p

theorem emma_knows_music (hEmma : ¬ CanEntertain Emma) : KnowsMusicTheory Emma := by
  have h1 : ¬ CanPlaySongs Emma := fun h => hEmma (A2 Emma h)
  have h2 : ¬ LearnedBasic Emma := fun h => h1 (A1 Emma h)
  have h3 : TookLessons Emma := by
    cases (A3 Emma) with
    | inl h => contradiction
    | inr h => exact h
  exact A4 Emma h3
</lean>

ANSWER: Yes
