The context provides several logical statements about the situation. Let's formalize them in Lean:

- `M`: It is Monday.
- `Q`: There is a quiz.
- `W`: Alex does well on the quiz.
- `S`: Alex studied hard.
- `G`: Alex is good at math.

Axioms:
1. If it is Monday, then there is a quiz: `M → Q`.
2. If Alex does well on the quiz, then they studied hard: `W → S`.
3. Either it is Monday, or Alex did not study hard: `M ∨ ¬S` (the third disjunct is redundant).
4. If Alex is good at math, then they do well on quizzes: `G → W`.

The question asks: If there was no quiz (`¬Q`), is Alex good at math (`G`)? We need to prove or disprove `¬Q → G`.

From the axioms:
- Assume `¬Q` (no quiz).
- By axiom 1 (`M → Q`), contraposition gives `¬Q → ¬M`, so `¬M`.
- By axiom 3 (`M ∨ ¬S`), since `¬M`, we must have `¬S` (disjunctive syllogism).
- By axiom 2 (`W → S`), contraposition gives `¬S → ¬W`, so `¬W`.
- By axiom 4 (`G → W`), contraposition gives `¬W → ¬G`, so `¬G`.

Thus, under `¬Q`, we derive `¬G`, meaning Alex is not good at math. Therefore, we have proven `¬Q → ¬G`, which is the negation of `¬Q → G`. Hence, the answer is **No**.

axiom M : Prop   -- It is Monday
axiom Q : Prop   -- There is a quiz

axiom G : Prop   -- Alex is good at math

axiom A1 : M → Q

axiom A3 : M ∨ ¬S
axiom A4 : G → W

theorem no_quiz_implies_not_good : ¬Q → ¬G := by
  intro hnoq
  intro hG
  have hW : W := A4 hG

  | inl hM => contradiction
  | inr hnS => exact hnS hS
