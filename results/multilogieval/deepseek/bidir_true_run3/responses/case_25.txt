The context provides several relationships:
- Building rapport leads to trusting relationships.
- Effective presentations convert sales.
- Not every salesperson builds trust (interpreted as building rapport) or converts sales.
- If a salesperson is perceptive, they build rapport.

Given that Emma gives effective presentations, we know she converts sales. However, there is no direct link between effective presentations (or converting sales) and being perceptive. The context only states that perceptive salespeople build rapport, but it does not connect presentation skills to perceptiveness. Additionally, the existence of salespeople who neither build rapport nor convert sales does not provide information about Emma's perceptiveness, as she converts sales and thus avoids one part of that negative category.

Therefore, we cannot prove that Emma is perceptive from the given premises.

<lean>
axiom Salesperson : Type
axiom Emma : Salesperson
axiom BuildsRapport : Salesperson → Prop
axiom HasTrust : Salesperson → Prop
axiom EffectivePresentations : Salesperson → Prop
axiom ConvertsSales : Salesperson → Prop
axiom Perceptive : Salesperson → Prop

-- Building rapport leads to trusting relationships
axiom A1 : ∀ s, BuildsRapport s → HasTrust s

-- Effective presentations convert sales
axiom A2 : ∀ s, EffectivePresentations s → ConvertsSales s

-- Not every salesperson builds trust (rapport) or converts sales: 
-- There exists a salesperson who does not build rapport and does not convert sales.
axiom A3 : ∃ s, ¬BuildsRapport s ∧ ¬ConvertsSales s

-- If perceptive, then build rapport
axiom A4 : ∀ s, Perceptive s → BuildsRapport s

-- Emma gives effective presentations
axiom T1 : EffectivePresentations Emma

-- Emma converts sales (from A2 and T1)
theorem Emma_converts : ConvertsSales Emma := by
  apply A2 Emma T1

-- We cannot prove Perceptive Emma because:
-- 1. EffectivePresentations only implies ConvertsSales, not Perceptive or BuildsRapport.
-- 2. A3 shows some salesperson fails both, but Emma converts sales so she avoids one failure.
-- 3. No axiom links EffectivePresentations/ConvertsSales to Perceptive.
</lean>

ANSWER: Failure
