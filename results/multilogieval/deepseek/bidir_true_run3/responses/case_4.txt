The reasoning involves proving that production costs increased given the premises. The context establishes:
- Investing in marketing (M) leads to more customers and increased sales (S): ∀q, M q → S q.
- For any quarter, either marketing focus (M), production cost increase (C), or both occurred: ∀q, M q ∨ C q.
- Sales did not increase last quarter: ¬S last_quarter.

From ¬S last_quarter and M last_quarter → S last_quarter, we deduce ¬M last_quarter (modus tollens). With M last_quarter ∨ C last_quarter and ¬M last_quarter, we conclude C last_quarter (disjunctive syllogism).

<lean>
axiom obj : Type
axiom last_quarter : obj
axiom Marketing : obj → Prop
axiom ProductionCostsIncreased : obj → Prop
axiom SalesIncreased : obj → Prop
axiom R1 : ∀ q : obj, Marketing q → SalesIncreased q
axiom R2 : ∀ q : obj, Marketing q ∨ ProductionCostsIncreased q
axiom not_S : ¬ SalesIncreased last_quarter

theorem production_costs_increased : ProductionCostsIncreased last_quarter := by
  have h_disj : Marketing last_quarter ∨ ProductionCostsIncreased last_quarter := R2 last_quarter
  have h_no_M : ¬ Marketing last_quarter := 
    fun hM : Marketing last_quarter => not_S (R1 last_quarter hM)
  cases h_disj with
  | inl hM => contradiction
  | inr hC => exact hC
</lean>
