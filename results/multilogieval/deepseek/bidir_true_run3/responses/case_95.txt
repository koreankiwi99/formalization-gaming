The reasoning involves proving that if Mary did not meet her friends for dinner, then it was an exciting night at the ballpark. The key steps are:

1. From premise 2: If Mary finishes errands, she meets friends. Thus, not meeting friends implies she did not finish errands (contrapositive).
2. Premise 3 states: Either weather clears up or errands are done (or both). Since errands weren't done, weather must clear up.
3. Premise 1: If weather clears up, the game goes on.
4. Premise 4: If game is on, fans cheer.
5. Premise 5: If fans cheer, it's an exciting night.

Thus, not meeting friends leads to an exciting night via this chain.

<lean>
axiom ClearUp : Prop
axiom FinishErrands : Prop
axiom GameOn : Prop
axiom MeetFriends : Prop
axiom Cheer : Prop
axiom ExcitingNight : Prop

axiom P1 : ClearUp → GameOn
axiom P2 : FinishErrands → MeetFriends
axiom P3 : ClearUp ∨ FinishErrands
axiom P4 : GameOn → Cheer
axiom P5 : Cheer → ExcitingNight

theorem result : ¬MeetFriends → ExcitingNight :=
  fun h : ¬MeetFriends =>
  have h_no_errands : ¬FinishErrands := fun herrands => h (P2 herrands)
  have h_clear : ClearUp :=
    Or.elim P3
      (fun clear => clear)
      (fun errands => absurd errands h_no_errands)
  have h_game : GameOn := P1 h_clear
  have h_cheer : Cheer := P4 h_game
  show ExcitingNight from P5 h_cheer
</lean>
