The reasoning involves proving that if the fans are not emotionally invested, then the atmosphere was electric. The premises are:

1. If home team wins (H), crowd is loud (L): H → L
2. If rivalry game (R), fans emotionally invested (E): R → E
3. Home team wins or rivalry game or both: H ∨ R
4. If crowd loud (L), atmosphere electric (A): L → A
5. If atmosphere electric (A), unforgettable game (U): A → U (irrelevant for this proof)

Given ¬E, we derive ¬R by contraposition of premise 2. With ¬R and H ∨ R, we get H. From H and premise 1, we get L. From L and premise 4, we get A. Thus, ¬E → A.

<lean>
axiom H : Prop
axiom R : Prop
axiom L : Prop
axiom E : Prop
axiom A : Prop

axiom P1 : H → L
axiom P2 : R → E
axiom P3 : H ∨ R
axiom P4 : L → A

theorem target : ¬E → A := by
  intro h
  have notR : ¬R := fun r => h (P2 r)
  have hH : H := by
    cases P3 with
    | inl h => exact h
    | inr r => contradiction
  have hL : L := P1 hH
  exact P4 hL
</lean>
