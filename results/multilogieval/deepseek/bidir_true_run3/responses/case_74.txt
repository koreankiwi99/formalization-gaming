The reasoning involves proving that James did not plan his schedule well, given that he is not going to the party. The context provides several logical relationships:

1. If it is Friday night, then every student goes to the party.
2. If a student finishes homework, they watch the game.
3. For every student, if they watch the game, it must be Friday night (from "Either it is Friday night or they are not watching the game").
4. If a student studies everyday, they finish homework.
5. If a student plans their schedule well, they study everyday.

Given that James is not going to the party:
- Since going to the party requires Friday night (from 1), and James isn't going, it cannot be Friday night.
- Because it isn't Friday night, no student can watch the game (from 3).
- James cannot watch the game, so he cannot have finished homework (from 2).
- Without finishing homework, James cannot have studied everyday (from 4).
- Without studying everyday, James could not have planned his schedule well (from 5).

This chain of implications proves that James did not plan his schedule well.

<lean>
axiom Person : Type
axiom FridayNight : Prop
axiom GoToParty : Person → Prop
axiom FinishHomework : Person → Prop
axiom WatchGame : Person → Prop
axiom StudyEveryday : Person → Prop
axiom PlanScheduleWell : Person → Prop

axiom R1 : ∀ (p : Person), FridayNight → GoToParty p
axiom R2 : ∀ (p : Person), FinishHomework p → WatchGame p
axiom R3 : ∀ (p : Person), WatchGame p → FridayNight
axiom R4 : ∀ (p : Person), StudyEveryday p → FinishHomework p
axiom R5 : ∀ (p : Person), PlanScheduleWell p → StudyEveryday p

axiom James : Person
axiom JamesNotGoing : ¬ GoToParty James

theorem JamesNotPlan : ¬ PlanScheduleWell James :=
  fun hPlan : PlanScheduleWell James =>
  have hStudy : StudyEveryday James := R5 James hPlan
  have hFinish : FinishHomework James := R4 James hStudy
  have hWatch : WatchGame James := R2 James hFinish
  have hFriday : FridayNight := R3 James hWatch
  have hGoing : GoToParty James := R1 James hFriday
  JamesNotGoing hGoing
