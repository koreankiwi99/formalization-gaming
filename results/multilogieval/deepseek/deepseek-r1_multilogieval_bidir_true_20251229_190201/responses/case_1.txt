The context provides logical implications and a disjunction:
1. Strong perimeter defense (P) forces many turnovers (T): P → T
2. Protecting the paint (B) limits inside scoring (L): B → L
3. Either the perimeter defense is strong or the paint is protected: P ∨ B

The query asks: If the Lakers did not limit inside scoring (¬L), did they force many turnovers (T)?  
Logically, prove ¬L → T.

Proof using modus tollens and disjunctive syllogism:
- From ¬L and B → L, conclude ¬B (contraposition).
- From ¬B and P ∨ B, conclude P.
- From P and P → T, conclude T.

theorem lakers_case : ¬L → T := by

    intro assumeB          -- Assume B (to derive contradiction)
    have L_holds : L := A2 assumeB   -- Using A2: B → L
    exact noL L_holds      -- Contradiction between ¬L and L
  cases A3 with           -- Case split on P ∨ B
  | inl p => exact A1 p   -- If P, get T from A1
  | inr b => contradiction  -- If B, contradicts ¬B
