The reasoning is translated into Lean using propositional logic. The premises are:
1. Watering regularly (W) implies blooming (B): `W → B`
2. Heavy rain (R) implies saturated soil (S): `R → S`
3. Either watered routinely or not saturated: `W ∨ ¬S`
4. Blooming implies fragrance (F): `B → F`
5. Fragrance attracts butterflies (A): `F → A`

The goal is to prove that heavy rain leads to butterflies: `R → A`.

Proof:
1. Assume heavy rain (R).
2. By premise 2, soil is saturated (S).
3. Premise 3: W ∨ ¬S. Since S is true, ¬S is false. Thus, W must be true (by disjunctive syllogism: S implies not ¬S, so W holds).
4. By premise 1, W → B, so blooming occurs (B).
5. By premise 4, B → F, so fragrance is present (F).
6. By premise 5, F → A, so butterflies are attracted (A).

Thus, heavy rain leads to butterflies: R → A.

axiom W : Prop -- garden watered routinely
axiom R : Prop -- heavy rain
axiom S : Prop -- soil saturated
axiom B : Prop -- flowers bloom
axiom F : Prop -- release fragrance
axiom A : Prop -- attract butterflies

axiom A1 : W → B
axiom A2 : R → S
axiom A3 : W ∨ ¬S
axiom A4 : B → F
axiom A5 : F → A

theorem attracts_butterflies : R → A := by
  intro hR 
  have hS : S := A2 hR 
  cases A3 with 
  | inl hW => 
      have hB : B := A1 hW 
      have hF : F := A4 hB 
      exact A5 hF 
  | inr hnS => 
      contradiction
