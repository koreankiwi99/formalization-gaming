The reasoning involves proving that if it was very hot outside, then the team won. The premises establish logical connections between practicing hard, preparation, game postponement, playing well, and winning. Assuming very hot leads to game postponement (premise 2). Premise 3 states that either the team practiced hard or the game wasn't postponed. Since postponement occurred, the team must have practiced hard (by disjunctive syllogism). Then, practicing hard implies preparation (premise 1), preparation implies playing well (premise 4), and playing well implies winning (premise 5). Thus, very hot implies win.

<lean>
axiom very_hot : Prop
axiom practices_hard : Prop
axiom postponed : Prop
axiom prepared : Prop
axiom plays_well : Prop
axiom win : Prop

-- Premises
axiom T1 : practices_hard → prepared
axiom T2 : very_hot → postponed
axiom T3 : practices_hard ∨ ¬postponed
axiom T4 : prepared → plays_well
axiom T5 : plays_well → win

theorem hot_implies_win : very_hot → win :=
  fun hHot : very_hot =>
    have hPost : postponed := T2 hHot
    have hHard : practices_hard := by
      cases T3 with
      | inl h => exact h
      | inr nh => exact absurd hPost nh
    have hPrep : prepared := T1 hHard
    have hPlay : plays_well := T4 hPrep
    show win from T5 hPlay
</lean>
