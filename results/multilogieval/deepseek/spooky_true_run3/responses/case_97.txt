The reasoning involves proving that if the kids are not hyper in the afternoon, then the students have lots of homework. Using the premises:

1. Substitute teacher → goof off (`P1`)
2. Pizza → hyper (`P2`)
3. Substitute ∨ pizza (`P3`)
4. Goof off → not learn much (`P4`)
5. Not learn much → lots of homework (`P5`)

Assume ¬hyper. By contraposition of `P2`, ¬hyper → ¬pizza. Thus, ¬pizza. From `P3` (sub ∨ pizza) and ¬pizza, substitute must be true. Then:
- Substitute → goof off (by `P1`)
- Goof off → not learn much (by `P4`)
- Not learn much → lots of homework (by `P5`)

Therefore, ¬hyper → lots of homework.

<lean>
axiom SubstituteTeacher : Prop
axiom Pizza : Prop
axiom GoofOff : Prop
axiom Hyper : Prop
axiom NotLearnMuch : Prop
axiom LotsOfHomework : Prop

axiom P1 : SubstituteTeacher → GoofOff
axiom P2 : Pizza → Hyper
axiom P3 : SubstituteTeacher ∨ Pizza
axiom P4 : GoofOff → NotLearnMuch
axiom P5 : NotLearnMuch → LotsOfHomework

theorem target : ¬Hyper → LotsOfHomework :=
  fun notHyper : ¬Hyper =>
  have notPizza : ¬Pizza :=
    fun hPizza : Pizza =>
    show False from notHyper (P2 hPizza)
  show LotsOfHomework from
    match P3 with
    | Or.inl sub =>
      have goof : GoofOff := P1 sub
      have notLearn : NotLearnMuch := P4 goof
      P5 notLearn
    | Or.inr pizza => False.elim (notPizza pizza)
</lean>
