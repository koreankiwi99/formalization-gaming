The premises lead to a contradiction when assuming no noise, which allows deriving any conclusion, including the police being called.

<lean>
axiom Noise : Prop
axiom Party : Prop
axiom Drums : Prop
axiom Complain : Prop
axiom Police : Prop

axiom P1 : Party → Noise
axiom P2 : Drums → Noise
axiom P3 : Party ∨ Drums
axiom P4 : Party → Complain
axiom P5 : Complain → Police

theorem result : ¬Noise → Police :=
  fun h : ¬Noise =>
    have no_party : ¬Party := fun p => h (P1 p)
    have no_drums : ¬Drums := fun d => h (P2 d)
    have contra : False := Or.elim P3 no_party no_drums
    False.elim contra
</lean>

ANSWER: Yes
