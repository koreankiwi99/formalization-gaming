The reasoning involves Alex, a student in the town. The premises indicate that for Alex:
- Studying hard implies good grades.
- Participating in extracurricular activities implies making new friends.
- Alex either studied hard or participated in extracurricular activities.
- Making new friends implies being invited to social events.

Given that Alex did not get good grades, we deduce:
1. Since studying hard leads to good grades (premise 1), and Alex lacks good grades, Alex did not study hard (contrapositive).
2. Therefore, from the disjunction (premise 3), Alex must have participated in extracurricular activities.
3. Participation implies making new friends (premise 2), so Alex made new friends.
4. Making new friends implies being invited to social events (premise 4), so Alex gets invited.

Thus, the conclusion follows.

axiom obj : Type
axiom Alex : obj
axiom StudyHard : obj → Prop
axiom GoodGrades : obj → Prop
axiom Participate : obj → Prop
axiom MakeFriends : obj → Prop
axiom Invited : obj → Prop

-- Premises:
axiom T1 : StudyHard Alex → GoodGrades Alex
axiom T2 : Participate Alex → MakeFriends Alex
axiom T3 : StudyHard Alex ∨ Participate Alex
axiom R1 : ∀ x : obj, MakeFriends x → Invited x

-- Given fact:
axiom T4 : ¬ GoodGrades Alex

theorem alex_invited : Invited Alex := by
  have no_study : ¬ StudyHard Alex := by
    intro h
    apply T4
    exact T1 h
  have participate_Alex : Participate Alex := by
    cases T3 with
    | inl h => contradiction
    | inr h => exact h
  have make_friends : MakeFriends Alex := T2 participate_Alex
  exact R1 Alex make_friends
</lean>
