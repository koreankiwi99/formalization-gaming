<lean>
axiom heavy_rain : Prop
axiom soil_saturated : Prop
axiom regular_watering : Prop
axiom blooms : Prop
axiom fragrance : Prop
axiom attract_butterflies : Prop

axiom A1 : regular_watering → blooms
axiom A2 : heavy_rain → soil_saturated
axiom A3 : regular_watering ∨ ¬soil_saturated
axiom A4 : blooms → fragrance
axiom A5 : fragrance → attract_butterflies

theorem rain_attracts_butterflies : heavy_rain → attract_butterflies := by
  intro hr
  have sat := A2 hr
  have rw : regular_watering := by
    cases A3 with
    | inl h => exact h
    | inr nh => contradiction
  have b := A1 rw
  have f := A4 b
  exact A5 f
</lean>

**Proof explanation:**  
1. Assume heavy rain (`hr`).  
2. By `A2`, soil becomes saturated (`sat`).  
3. By `A3` (disjunction):  
   - If `regular_watering` holds, use it directly.  
   - If `¬soil_saturated` holds, it contradicts `sat`, proving `regular_watering` by contradiction.  
4. `regular_watering` implies `blooms` (`A1`), which implies `fragrance` (`A4`), which implies `attract_butterflies` (`A5`).  

Thus, heavy rain necessarily leads to butterflies.
