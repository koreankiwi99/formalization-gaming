The premises state that Susan flies to LGA, John flies from LGA, and no flight can have the same departure and arrival airport. The statement claims John flies to LGA. If John flies to LGA, then his flight would depart from LGA and arrive at LGA, violating the premise that departure and arrival cannot be the same. This contradiction proves the statement false.

<lean>
axiom Person : Type
axiom Airport : Type
axiom LGA : Airport
axiom Susan : Person
axiom John : Person
axiom fliesTo : Person → Airport → Prop
axiom fliesFrom : Person → Airport → Prop

axiom susan_to : fliesTo Susan LGA
axiom rule : ∀ (p : Person) (a b : Airport), fliesFrom p a → fliesTo p b → a ≠ b
axiom john_from : fliesFrom John LGA

theorem disprove_statement : ¬ fliesTo John LGA :=
  fun h => (rule John LGA LGA john_from h) (rfl)

</lean>

ANSWER: False
