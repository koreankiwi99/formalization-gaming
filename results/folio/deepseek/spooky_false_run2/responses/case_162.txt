The premises state that no hawk swims and that there exists at least one bird that is a hawk. This implies that there exists a bird (the hawk) that does not swim. Therefore, the statement "All birds swim" must be false, as there is a counterexample.

<lean>
axiom obj : Type
axiom Bird : obj → Prop
axiom Hawk : obj → Prop
axiom Swims : obj → Prop

axiom T1 : ∀ x, Hawk x → ¬ Swims x
axiom T2 : ∃ x, Bird x ∧ Hawk x

theorem not_all_birds_swim : ¬ (∀ x, Bird x → Swims x) := by
  intro H
  obtain ⟨w, hBird, hHawk⟩ := T2
  have no_swim : ¬ Swims w := T1 w hHawk
  have swim : Swims w := H w hBird
  contradiction
</lean>

ANSWER: False
