The premises lead to a contradiction if we assume "Black Mirror" is popular, proving it cannot be popular.

<lean>
axiom obj : Type
axiom StrangerThings : obj
axiom BlackMirror : obj
axiom Popular : obj → Prop
axiom BingeWatches : obj → Prop
axiom Downloads : obj → Prop
axiom SharesWithLisa : obj → Prop

axiom T1 : Popular StrangerThings
axiom R1 : ∀ s : obj, Popular s → BingeWatches s
axiom R2 : ∀ s : obj, BingeWatches s ↔ Downloads s
axiom T2 : ¬ Downloads BlackMirror
axiom T3 : ∀ s : obj, BingeWatches s → SharesWithLisa s

theorem black_mirror_not_popular : ¬ Popular BlackMirror := by
  intro h
  have binge_BM : BingeWatches BlackMirror := R1 BlackMirror h
  have dl_BM : Downloads BlackMirror := (R2 BlackMirror).mp binge_BM
  exact T2 dl_BM
</lean>

ANSWER: False
