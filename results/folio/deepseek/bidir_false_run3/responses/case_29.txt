The logical reasoning is translated into Lean 4, where predicates and entities are defined, and the axioms are formalized. The goal is to prove that the statement "If Marvin is not from Mars, then Marvin is a human" is false. This statement is represented as the implication `¬FromMars Marvin → Human Marvin`. Its negation is `¬(¬FromMars Marvin → Human Marvin)`, which is equivalent to `¬FromMars Marvin ∧ ¬Human Marvin`.

The proof proceeds as follows:
1. The key axiom `A6` splits into two cases: either Marvin is from both Earth and Mars, or from neither.
2. Case 1 (Marvin is from both) leads to a contradiction:
   - Being from Mars implies being an alien (by `A2`), hence an extraterrestrial (by `A1`).
   - Being an extraterrestrial implies not being human (by `A3`).
   - However, being a highly intelligent being from Earth (by `A4` and `A5`) implies being human.
   - This contradiction disproves Case 1.
3. Therefore, Case 2 must hold: Marvin is from neither Earth nor Mars.
4. Using `A7`, since Marvin is not from Earth, he is an extraterrestrial.
   - By `A3`, extraterrestrials are not human, so Marvin is not human.
5. In this case, `¬FromMars Marvin` is true, but `Human Marvin` is false, making the implication `¬FromMars Marvin → Human Marvin` false.

The Lean code formalizes these steps. The theorem `statement_false` proves the negation of the target statement, confirming it is false.

<lean>
axiom Entity : Type

axiom Marvin : Entity
axiom Alien : Entity → Prop
axiom Extraterrestrial : Entity → Prop
axiom FromMars : Entity → Prop
axiom FromEarth : Entity → Prop
axiom Human : Entity → Prop
axiom HighlyIntelligent : Entity → Prop

axiom A1 : ∀ x : Entity, Alien x → Extraterrestrial x
axiom A2 : ∀ x : Entity, FromMars x → Alien x
axiom A3 : ∀ x : Entity, Extraterrestrial x → ¬ Human x
axiom A4 : ∀ x : Entity, HighlyIntelligent x ∧ FromEarth x → Human x
axiom A5 : HighlyIntelligent Marvin
axiom A6 : (FromEarth Marvin ∧ FromMars Marvin) ∨ (¬ FromEarth Marvin ∧ ¬ FromMars Marvin)
axiom A7 : ¬ FromEarth Marvin → Extraterrestrial Marvin

theorem statement_false : ¬(¬FromMars Marvin → Human Marvin) :=
  by
    intro imp
    cases A6 with
    | inl case1 =>
        have human : Human Marvin := A4 Marvin ⟨A5, case1.left⟩
        have alien : Alien Marvin := A2 Marvin case1.right
        have et : Extraterrestrial Marvin := A1 Marvin alien
        have not_human : ¬Human Marvin := A3 Marvin et
        exact not_human human
    | inr case2 =>
        have et : Extraterrestrial Marvin := A7 case2.left
        have not_human : ¬Human Marvin := A3 Marvin et
        exact not_human (imp case2.right)
