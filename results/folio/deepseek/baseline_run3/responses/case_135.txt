<lean>
axiom obj : Type
axiom Suck : obj → Prop
axiom Plunger : obj → Prop
axiom Vacuum : obj → Prop
axiom Vampire : obj → Prop
axiom Duster : obj → Prop
axiom Space : obj

axiom plungers_suck : ∀ x : obj, Plunger x → Suck x
axiom vacuums_suck : ∀ x : obj, Vacuum x → Suck x
axiom vampires_suck : ∀ x : obj, Vampire x → Suck x
axiom space_vacuum : Vacuum Space
axiom duster_no_suck : ∀ x : obj, Duster x → ¬Suck x

-- We want to prove or disprove: Vampire Space

-- From space_vacuum and vacuums_suck, we know Suck Space
theorem space_sucks : Suck Space := vacuums_suck Space space_vacuum

-- However, we have no axiom stating that sucking implies being a vampire
-- We also have no information that vacuums cannot be vampires
-- The existence of dusters (which don't suck) doesn't constrain Space
-- Therefore, Vampire Space is neither provable nor disprovable
</lean>

ANSWER: Uncertain
