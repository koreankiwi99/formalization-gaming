The context provides information about animals spotted near the campus, including Rockie. The axioms establish:
- Rockie is spotted and calm.
- Spotted animals are either rabbits or squirrels.
- If Rockie were a squirrel, it would be skittish (by axiom 5), leading to not calm (by axiom 4), contradicting Rockie's calmness. Thus, Rockie must be a rabbit.
- However, there is no axiom stating that rabbits cannot be turtles. Axiom 2 confirms some turtles are spotted, but it does not specify whether turtles are distinct from rabbits or if Rockie is among them. Therefore, while Rockie is a rabbit, it remains uncertain whether Rockie is also a turtle or not.

<lean>
axiom obj : Type
axiom Rockie : obj
axiom Rabbit : obj → Prop
axiom Turtle : obj → Prop
axiom Squirrel : obj → Prop
axiom SpottedNearCampus : obj → Prop
axiom Cute : obj → Prop
axiom Skittish : obj → Prop
axiom Calm : obj → Prop

axiom A1 : ∀ x : obj, Rabbit x ∧ SpottedNearCampus x → Cute x
axiom A2 : ∃ x : obj, Turtle x ∧ SpottedNearCampus x
axiom A3 : ∀ x : obj, SpottedNearCampus x → Rabbit x ∨ Squirrel x
axiom A4 : ∀ x : obj, Skittish x → ¬ Calm x
axiom A5 : ∀ x : obj, Squirrel x ∧ SpottedNearCampus x → Skittish x
axiom A6 : SpottedNearCampus Rockie ∧ Calm Rockie

-- Prove Rockie is a rabbit (not a squirrel)
theorem Rockie_is_rabbit : Rabbit Rockie := by
  have h_spot := A6.left
  have h_calm := A6.right
  cases A3 Rockie h_spot with
  | inl h_rabbit => exact h_rabbit
  | inr h_squirrel =>
      have h_skittish : Skittish Rockie := A5 Rockie ⟨h_squirrel, h_spot⟩
      have h_not_calm := A4 Rockie h_skittish
      contradiction
