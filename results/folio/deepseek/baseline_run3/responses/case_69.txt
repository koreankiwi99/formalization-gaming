The statement "No members of Pappy's have starred in a show piloting on BBC Two or BBC Three" is false. The context indicates that Badults, which starred members of Pappy's, piloted on BBC Three. This directly contradicts the statement, as it demonstrates that at least one member of Pappy's starred in a show (Badults) that piloted on BBC Three.

<lean>
axiom Person : Type
axiom Show : Type
axiom Channel : Type

axiom BBC_Two : Channel
axiom BBC_Three : Channel

axiom badults : Show
axiom Pappys : Person → Prop
axiom starred_in : Person → Show → Prop
axiom pilot_channel : Show → Channel → Prop

axiom piloted_Badults_Three : pilot_channel badults BBC_Three
axiom Pappys_star_Badults : ∃ p, Pappys p ∧ starred_in p badults

theorem statement_false : 
    ¬(∀ p, Pappys p → ∀ s, starred_in p s → 
      ¬(pilot_channel s BBC_Two ∨ pilot_channel s BBC_Three)) :=
by
  intro h
  rcases Pappys_star_Badults with ⟨p, hp, hstar⟩
  have no_pilot := h p hp badults hstar
  apply no_pilot
  apply Or.inr
  exact piloted_Badults_Three
</lean>

ANSWER: False
