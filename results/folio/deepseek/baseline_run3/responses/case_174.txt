   If Taylor grew up in a health-conscious home (H) and disregards her well-being (D):  
   - By A2, H implies she eats salads regularly (S).  
   - By A1, S implies she is health-conscious (C).  
   - By A4, D implies she is not health-conscious (¬C).  
   This creates a contradiction (C ∧ ¬C), making this case impossible.

2. **Deduction in Case 2**:  
   Thus, Taylor must be in the second case: ¬H ∧ ¬D.  
   - We have ¬H.  
   - To show ¬G (she doesn't visit the gym daily):  
     Assume G. By A5, G implies she fulfills nutrition (F).  
     By A3, F implies H, contradicting ¬H.  
   Hence ¬G holds.  

3. **Conclusion**:  
   Therefore, ¬H ∧ ¬G is true.

axiom T : obj

-- Predicates
axiom SaladsRegularly : obj → Prop
axiom HealthConscious : obj → Prop
axiom HealthConsciousChildhood : obj → Prop
axiom FulfillNutrition : obj → Prop
axiom DisregardWellBeing : obj → Prop

-- Axioms

axiom A2 : ∀ x : obj, HealthConsciousChildhood x → SaladsRegularly x
axiom A3 : ∀ x : obj, FulfillNutrition x → HealthConsciousChildhood x
axiom A4 : ∀ x : obj, DisregardWellBeing x → ¬ HealthConscious x
axiom A5 : ∀ x : obj, VisitGymDaily x → FulfillNutrition x

-- Taylor's disjunction
axiom A6 : (HealthConsciousChildhood T ∧ DisregardWellBeing T) ∨ (¬ HealthConsciousChildhood T ∧ ¬ DisregardWellBeing T)

-- Theorem: Taylor neither grew up in a health-conscious childhood home nor visits the gym daily.
theorem taylor_case : ¬ HealthConsciousChildhood T ∧ ¬ VisitGymDaily T := by
  cases A6 with
  | inl h =>

      have H := h.left
      have D := h.right

      have notC : ¬ HealthConscious T := A4 T D
