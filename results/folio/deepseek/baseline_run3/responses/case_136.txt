The context provides that a duster is a household appliance that does not suck. This directly contradicts the statement "If something is a household appliance, it sucks," as we have a counterexample (the duster). Therefore, the statement is false.

<lean>
axiom obj : Type
axiom Suck : obj → Prop
axiom HouseholdAppliance : obj → Prop
axiom Duster : obj
axiom duster_appliance : HouseholdAppliance Duster
axiom duster_no_suck : ¬ Suck Duster

theorem household_suck_false : ¬ (∀ x, HouseholdAppliance x → Suck x) :=
  fun h => duster_no_suck (h Duster duster_appliance)
</lean>

ANSWER: False
