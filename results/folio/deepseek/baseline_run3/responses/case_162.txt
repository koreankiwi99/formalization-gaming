The context states: "A hawk never swims" and "Some birds are hawks." The question is whether "All birds swim" is true, false, or uncertain.

- "A hawk never swims" means: ∀x, Hawk x → ¬Swim x
- "Some birds are hawks" means: ∃x, Bird x ∧ Hawk x
- The statement "All birds swim" is: ∀x, Bird x → Swim x

From the axioms:
1. There exists some object (say a) that is both a bird and a hawk.
2. Since it's a hawk, it does not swim (by the first axiom).
3. Therefore, there exists a bird (namely a) that does not swim.
4. This directly contradicts the claim that "all birds swim."

Thus, "All birds swim" is false.

<lean>
axiom obj : Type
axiom Bird : obj → Prop
axiom Hawk : obj → Prop
axiom Swim : obj → Prop

axiom hawk_never_swims : ∀ x, Hawk x → ¬ Swim x
axiom some_birds_are_hawks : ∃ x, Bird x ∧ Hawk x

theorem not_all_birds_swim : ¬ (∀ x, Bird x → Swim x) := by
  intro h
  rcases some_birds_are_hawks with ⟨w, hw_bird, hw_hawk⟩
  have h_no_swim := hawk_never_swims w hw_hawk
  have h_swim := h w hw_bird
  contradiction
</lean>

ANSWER: False
