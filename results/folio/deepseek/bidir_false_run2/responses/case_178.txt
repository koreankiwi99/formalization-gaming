The reasoning involves proving that the statement "Some humans are horses" is false given the premises. The argument proceeds as follows:

1. **Premise 1**: All horses have hooves (∀x, Horse x → Hooves x)
2. **Premise 2**: No humans have hooves (∀x, Human x → ¬Hooves x)
3. **To disprove**: ∃x, Human x ∧ Horse x (some humans are horses)

**Proof by contradiction**:
- Assume there exists an entity `x` that is both human and horse.
- Since `x` is a horse (from 1st premise), `x` must have hooves.
- Since `x` is human (from 2nd premise), `x` cannot have hooves.
- This contradiction implies the assumption ∃x, Human x ∧ Horse x is false.

The Lean code formalizes this reasoning, proving the negation of the statement. Since the proof succeeds, we conclude "False".

axiom Horse : obj → Prop
axiom Human : obj → Prop
axiom Hooves : obj → Prop
axiom A1 : ∀ x : obj, Horse x → Hooves x

theorem disprove_statement : ¬ ∃ x : obj, Human x ∧ Horse x :=
