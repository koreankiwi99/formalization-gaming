The premises state that some birds are hawks and that hawks never swim. This implies that there exists at least one bird (a hawk) that does not swim, which contradicts the statement that all birds swim. Therefore, "All birds swim" is false.

<lean>
axiom obj : Type
axiom Bird : obj → Prop
axiom Hawk : obj → Prop
axiom Swims : obj → Prop

axiom T1 : ∀ (x : obj), Hawk x → ¬ Swims x
axiom T2 : ∃ (x : obj), Bird x ∧ Hawk x

theorem false_statement : ¬ (∀ (x : obj), Bird x → Swims x) := 
by
  intro H
  rcases T2 with ⟨w, hw_bird, hw_hawk⟩
  have contra := T1 w hw_hawk
  exact contra (H w hw_bird)
</lean>

The Lean proof demonstrates that the universal claim "all birds swim" leads to a contradiction given the premises. The witness `w` (which is both a bird and a hawk) cannot swim by `T1`, yet must swim under the assumption `H`, producing a contradiction. This proves the original statement false.

ANSWER: False
