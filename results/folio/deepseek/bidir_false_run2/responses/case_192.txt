The context states that events are either happy or sad, which implies that happy and sad are mutually exclusive (an event cannot be both). Additionally, at least one event is happy. The statement "All events are sad" leads to a contradiction because the happy event cannot be sad at the same time. Therefore, we can refute the statement.

We formalize this in Lean:
- The type `Event` represents events.
- `Happy e` and `Sad e` are predicates indicating whether an event is happy or sad.
- The existence of a happy event is given as `∃ e, Happy e`.
- Mutual exclusion is stated as `∀ e, Happy e → ¬ Sad e`.
- We prove that "all events are sad" is false: `¬ (∀ e, Sad e)` by showing the happy event must be sad under this assumption, contradicting mutual exclusion.

<lean>
axiom Event : Type
axiom Happy : Event → Prop
axiom Sad : Event → Prop
axiom T1 : ∃ e, Happy e
axiom R1 : ∀ e, Happy e → ¬ Sad e
theorem refutation : ¬ (∀ (e : Event), Sad e) := 
  fun h_all_sad =>
    let ⟨e, happy_e⟩ := T1
    R1 e happy_e (h_all_sad e)
</lean>

We have successfully proven the statement false.

ANSWER: False
