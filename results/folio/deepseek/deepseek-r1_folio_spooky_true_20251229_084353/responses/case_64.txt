The reasoning is logically consistent. The premises establish that Ted, being an alligator, cannot be a domesticated animal. However, if Ted were a Brown Swiss cattle, this would imply Ted is a cow (via universal quantification), and thus a domesticated animal. This leads to a contradiction. Therefore, assuming Ted is a Brown Swiss cattle implies any statement, including that Ted is not a pet.

<lean>
axiom obj : Type
axiom Ted : obj
axiom BrownSwiss : obj → Prop
axiom Cow : obj → Prop
axiom Pet : obj → Prop
axiom DomesticatedAnimal : obj → Prop
axiom Alligator : obj → Prop

axiom all_brown_swiss_are_cows : ∀ x, BrownSwiss x → Cow x
axiom all_cows_domesticated : ∀ x, Cow x → DomesticatedAnimal x
axiom alligators_not_domesticated : ∀ x, Alligator x → ¬ DomesticatedAnimal x
axiom Ted_alligator : Alligator Ted

theorem condition : BrownSwiss Ted → ¬ Pet Ted :=
  fun h_bs _ =>
    have h_cow : Cow Ted := all_brown_swiss_are_cows Ted h_bs
    have h_dom : DomesticatedAnimal Ted := all_cows_domesticated Ted h_cow
    alligators_not_domesticated Ted Ted_alligator h_dom
