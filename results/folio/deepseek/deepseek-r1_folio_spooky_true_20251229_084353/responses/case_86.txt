<lean>
axiom obj : Type
axiom sir_digby : obj
axiom nemesis : obj
axiom SuperheroMovie : obj → Prop
axiom NamedAfter : obj → obj → Prop
axiom GoodGuy : obj → Prop
axiom BadGuy : obj → Prop
axiom Fights : obj → obj → Prop
axiom Wins : obj → Prop

-- Specific movie (sir_digby_movie) and character relations
axiom movie : obj
axiom is_superhero : SuperheroMovie movie
axiom named_digby : NamedAfter movie sir_digby
axiom fight_relation : Fights sir_digby nemesis

-- Premises
axiom superhero_win_rule : ∀ m c, SuperheroMovie m → GoodGuy c → Wins c
axiom named_good : ∀ m c, SuperheroMovie m → NamedAfter m c → GoodGuy c
axiom fight_win_rule : ∀ c1 c2, Fights c1 c2 → (Wins c1 → ¬ Wins c2) ∧ (Wins c2 → ¬ Wins c1)
axiom fight_polarity : ∀ c1 c2, Fights c1 c2 → 
  (GoodGuy c1 ∧ BadGuy c2) ∨ (GoodGuy c2 ∧ BadGuy c1)

-- Derived facts
theorem digby_good : GoodGuy sir_digby := 
  named_good movie sir_digby is_superhero named_digby

theorem digby_wins : Wins sir_digby := 
  superhero_win_rule movie sir_digby is_superhero digby_good

theorem fight_rule : Wins sir_digby → ¬ Wins nemesis := 
  (fight_win_rule sir_digby nemesis fight_relation).1

theorem nemesis_loss : ¬ Wins nemesis := 
  fight_rule digby_wins
</lean>

**Proof explanation**:
1. `digby_good` follows from the movie being superhero and named after Sir Digby
2. `digby_wins` uses the superhero movie rule applied to Sir Digby
3. `fight_rule` extracts the directional win principle from the fight
4. `nemesis_loss` combines both results to prove the conclusion
