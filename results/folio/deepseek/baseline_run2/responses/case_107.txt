The context provides several implications and a disjunction about Peter:

1. Spills food → ¬notably tidy
2. Clumsy foodies who go out frequently → spill food
3. Cleanly → notably tidy
4. Value order → cleanly
5. Family prioritizes order → value order
6. Peter: (spills food ∧ cleanly) ∨ (¬spills food ∧ ¬cleanly)

The disjunction in (6) must be evaluated:
- Case 1 (spills ∧ cleanly):
  - From (3): cleanly → notably tidy
  - From (1): spills → ¬notably tidy
  - Contradiction: notably tidy and ¬notably tidy → impossible
- Thus, only case 2 is possible: ¬spills food ∧ ¬cleanly

Now, to prove "Peter is notably tidy":
- We have ¬cleanly Peter (from case 2)
- Axiom (3) states cleanly → notably tidy, but the converse isn't given
- No other axioms force notably tidy from ¬cleanly or provide alternative paths to notably tidy
- Conversely, no axioms prove Peter isn't notably tidy (spills is false, so (1) doesn't apply; no other relations prohibit tidiness)

Since we cannot prove Peter is notably tidy nor prove he isn't, the statement is uncertain.

<lean>
axiom obj : Type
axiom Peter : obj
axiom SpillsFood : obj → Prop
axiom NotablyTidy : obj → Prop
axiom Cleanly : obj → Prop
axiom ValuesOrder : obj → Prop
axiom FamilyPrioritizesOrder : obj → Prop
axiom ClumsyFoodie : obj → Prop
axiom GoOutFrequently : obj → Prop

axiom spill_not_tidy : ∀ x, SpillsFood x → ¬NotablyTidy x
axiom clumsy_spill : ∀ x, ClumsyFoodie x ∧ GoOutFrequently x → SpillsFood x
axiom cleanly_tidy : ∀ x, Cleanly x → NotablyTidy x
axiom value_clean : ∀ x, ValuesOrder x → Cleanly x
axiom family_value : ∀ x, FamilyPrioritizesOrder x → ValuesOrder x
axiom peter_cases : (SpillsFood Peter ∧ Cleanly Peter) ∨ (¬SpillsFood Peter ∧ ¬Cleanly Peter)

theorem case1_contradiction : ¬(SpillsFood Peter ∧ Cleanly Peter) := by
  intro ⟨spill_peter, clean_peter⟩
  have tidy_peter : NotablyTidy Peter := cleanly_tidy Peter clean_peter
  have not_tidy_peter : ¬NotablyTidy Peter := spill_not_tidy Peter spill_peter
  contradiction

theorem peter_final : ¬SpillsFood Peter ∧ ¬Cleanly Peter := by
  cases peter_cases with
  | inl h => exact absurd h case1_contradiction
  | inr h => exact h

-- NotablyTidy Peter cannot be proven or disproven:
-- - ¬Cleanly Peter (from peter_final) blocks cleanly_tidy
-- - No other axioms force NotablyTidy Peter
-- - ¬SpillsFood Peter blocks spill_not_tidy, but that only prevents ¬NotablyTidy Peter
