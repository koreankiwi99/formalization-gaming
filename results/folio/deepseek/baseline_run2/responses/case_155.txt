The context states that all humans are mortal and all Greeks are humans. The question asks whether it follows that some Greeks are mortal.

In logic, the argument is:
- Premise 1: ∀x, Human x → Mortal x  (All humans are mortal)
- Premise 2: ∀x, Greek x → Human x   (All Greeks are humans)
- Conclusion: ∃x, Greek x ∧ Mortal x (Some Greeks are mortal)

While the premises imply that *if* there is a Greek, then they are mortal:
  ∀x, Greek x → Mortal x
they do not guarantee the existence of a Greek. The conclusion requires at least one Greek to exist and be mortal. Without knowing whether there are any Greeks, the truth value of the conclusion cannot be determined:
- If there are Greeks, the conclusion is true.
- If there are no Greeks, the conclusion is false.

The premises do not provide information about the existence of Greeks, so the conclusion is uncertain.

-- The theorem to prove: ∃ x, Greek x ∧ Mortal x
-- We cannot prove it without knowing that at least one Greek exists.
example : ∃ x, Greek x ∧ Mortal x := 
  by 
    -- We have no object of type `obj` to work with
    -- Given an arbitrary `x`, we can derive properties conditionally,

    sorry   -- Not provable with given axioms

-- ¬∃ x, Greek x ∧ Mortal x  (meaning no Greek is mortal)
-- because if there were a Greek, they would be mortal by the axioms.

    sorry  -- Negation not provable
