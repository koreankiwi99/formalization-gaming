The reasoning context is translated into Lean using appropriate predicates and axioms. The proof demonstrates that Marvin cannot be both from Earth and Mars (leading to a contradiction) and must therefore be from neither, making him extraterrestrial (by axiom T7) and non-human (by T3). The disjunction about his origins (axiom T6) forces this conclusion.

<lean>
axiom obj : Type

axiom Alien : obj → Prop
axiom Extraterrestrial : obj → Prop
axiom FromMars : obj → Prop
axiom Human : obj → Prop
axiom HighlyIntelligent : obj → Prop
axiom FromEarth : obj → Prop

axiom Marvin : obj

axiom T1 : ∀ x, Alien x → Extraterrestrial x
axiom T2 : ∀ x, FromMars x → Alien x
axiom T3 : ∀ x, Extraterrestrial x → ¬ Human x
axiom T4 : ∀ x, (HighlyIntelligent x ∧ FromEarth x) → Human x
axiom T5 : HighlyIntelligent Marvin
axiom T6 : (FromEarth Marvin ∧ FromMars Marvin) ∨ (¬FromEarth Marvin ∧ ¬FromMars Marvin)
axiom T7 : ¬ FromEarth Marvin → Extraterrestrial Marvin

theorem result : ¬ Human Marvin ∧ ¬ FromMars Marvin := by
  cases T6 with
  | inl h => 
      have hmars := h.right
      have hhuman := T4 Marvin ⟨T5, h.left⟩
      have hnonhuman := T3 Marvin (T1 Marvin (T2 Marvin hmars))
      contradiction
  | inr h => 
      have hnonhuman := T3 Marvin (T7 h.left)
      exact ⟨hnonhuman, h.right⟩
</lean>

ANSWER: True
