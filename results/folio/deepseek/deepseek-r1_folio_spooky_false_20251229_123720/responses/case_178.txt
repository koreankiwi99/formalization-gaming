The premises state that all horses have hooves and no humans have hooves. The statement to evaluate is "some humans are horses." 

Using predicate logic:
1. ∀x (Horse(x) → HasHooves(x))
2. ∀x (Human(x) → ¬HasHooves(x))

The statement ∃x (Human(x) ∧ Horse(x)) leads to a contradiction. For any x that is both human and horse, we derive:
- HasHooves(x) (from being a horse)
- ¬HasHooves(x) (from being a human)

Thus, the existential claim is false. The Lean proof confirms this by showing that assuming such an x exists results in a contradiction between the derived hooves properties.

<lean>
axiom obj : Type
axiom Horse : obj → Prop
axiom Human : obj → Prop
axiom HasHooves : obj → Prop

axiom all_horses_have_hooves : ∀ x : obj, Horse x → HasHooves x
axiom no_humans_have_hooves : ∀ x : obj, Human x → ¬HasHooves x

theorem disprove_existential : ¬∃ x : obj, Human x ∧ Horse x := by
  intro h_exists
  rcases h_exists with ⟨w, hw_human, hw_horse⟩
  have hw_hooves : HasHooves w := all_horses_have_hooves w hw_horse
  have hw_no_hooves : ¬HasHooves w := no_humans_have_hooves w hw_human
  exact hw_no_hooves hw_hooves
</lean>

ANSWER: False
