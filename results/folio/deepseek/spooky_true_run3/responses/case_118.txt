The key challenge is interpreting the ambiguous phrase "Antarctica is located on the South Shetland Islands." axiom obj : Type

axiom Barutin : obj

axiom Antarctica : obj

axiom Transitivity : ∀ x y z, LocatedIn x y → LocatedIn y z → LocatedIn x z

theorem coves_in_antarctica : ∃ c place, Cove c ∧ LocatedIn c Antarctica ∧ NamedAfter c place ∧ BulgarianSettlement place := by
