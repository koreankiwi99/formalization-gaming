The context states that Yale University is a private Ivy League research university and that it moved to New Haven. The statement claims that a private Ivy League research university moved to New Haven, which is existentially quantified. Since Yale satisfies both properties, we can prove the existential claim.

<lean>
axiom obj : Type
axiom Yale : obj
axiom IsPrivateIvyResearch : obj → Prop
axiom MovedToNewHaven : obj → Prop

-- Context axioms
axiom yale_type : IsPrivateIvyResearch Yale
axiom yale_move : MovedToNewHaven Yale

-- Prove existential: ∃ x, IsPrivateIvyResearch x ∧ MovedToNewHaven x
theorem exists_ivy_moved : ∃ x, IsPrivateIvyResearch x ∧ MovedToNewHaven x :=
  by
    exists Yale
    exact ⟨yale_type, yale_move⟩
</lean>

ANSWER: True
