<lean>
axiom Animal : Type

axiom Invertebrate : Animal → Prop
axiom Vertebrate : Animal → Prop
axiom HasBackbone : Animal → Prop
axiom ReproducesByMating : Animal → Prop
axiom IsBee : Animal → Prop
axiom IsQueenBee : Animal → Prop

-- Animals are either invertebrates or vertebrates
axiom animal_classification : ∀ a : Animal, Invertebrate a ∨ Vertebrate a

-- All animals with backbones reproduce by male-and-female mating
axiom backbone_reproduction : ∀ a : Animal, HasBackbone a → ReproducesByMating a

-- All vertebrate animals have a backbone
axiom vertebrate_has_backbone : ∀ a : Animal, Vertebrate a → HasBackbone a

-- All bees do not reproduce by male-and-female mating
axiom bee_no_mating : ∀ a : Animal, IsBee a → ¬ ReproducesByMating a

-- All queen bees are bees
axiom queen_bee_is_bee : ∀ a : Animal, IsQueenBee a → IsBee a

-- Harry is a bee
axiom harry_bee : Animal
axiom harry_is_bee : IsBee harry_bee

-- We need to prove: Harry is an invertebrate or a queen bee
theorem harry_invertebrate_or_queen : Invertebrate harry_bee ∨ IsQueenBee harry_bee := by
  -- Suppose Harry were not an invertebrate
  apply Or.elim (animal_classification harry_bee)
  · intro h_inv
    exact Or.inl h_inv
  · intro h_vert
    -- If Harry is vertebrate, then he has a backbone
    have h_backbone : HasBackbone harry_bee := vertebrate_has_backbone harry_bee h_vert
    -- If he has a backbone, he must reproduce by mating
    have h_mating : ReproducesByMating harry_bee := backbone_reproduction harry_bee h_backbone
    -- But as a bee, Harry does not reproduce by mating
    have h_no_mating : ¬ ReproducesByMating harry_bee := bee_no_mating harry_bee harry_is_bee
    -- Contradiction
    contradiction
</lean>

The proof proceeds as follows:
1. We know Harry must be either invertebrate or vertebrate (animal_classification)
2. If he's invertebrate, the left side of the disjunction holds
3. If he's vertebrate:
   - Then he has a backbone (vertebrate_has_backbone)
   - Then he reproduces by mating (backbone_reproduction)
   - But bees (including Harry) don't reproduce by mating (bee_no_mating)
4. The contradiction shows Harry cannot be vertebrate, leaving only invertebrate

Thus we've proven `Invertebrate harry_bee`, making the disjunction true.

ANSWER: True
