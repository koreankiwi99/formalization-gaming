The context states that "KO is a mature stock," which implies that KO is both a stock and mature. This is formalized as `Stock KO ∧ MatureStock KO`. Therefore, `Stock KO` is directly provable from this conjunction.

<lean>
axiom obj : Type

axiom Stock : obj → Prop
axiom GrowthStock : obj → Prop
axiom MatureStock : obj → Prop
axiom BoughtForProfit : obj → Prop
axiom SuitableForRetirement : obj → Prop

axiom A1 : ∀ x, GrowthStock x → BoughtForProfit x
axiom A2 : ∀ x, BoughtForProfit x → ¬ SuitableForRetirement x
axiom A3 : ∃ x, Stock x ∧ GrowthStock x
axiom A4 : ∀ x, MatureStock x → SuitableForRetirement x
axiom A5 : Stock KO ∧ MatureStock KO  -- "KO is a mature stock" interpreted as conjunction

theorem ko_stock : Stock KO := by
