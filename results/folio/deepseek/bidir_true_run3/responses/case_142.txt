The context provides information about breeding back, Heck cattle, and artificial selection. The key premises are:
1. Breeding back is a form of artificial selection (∀x, BredBack x → ArtificialSelection x).
2. Heck cattle were bred back (∀x, HeckCattle x → BredBack x).
3. There exists at least one Heck cattle (∃x, HeckCattle x).

From these, we can prove that some Heck cattle are artificially selected:
- Take an individual Heck cattle (exists by axiom T2).
- Since it's Heck cattle, it was bred back (by axiom A2).
- Since it was bred back, it is artificially selected (by axiom A1).
- Thus, there exists a Heck cattle that is artificially selected.

<lean>
axiom obj : Type

axiom Animal : obj → Prop
axiom HeckCattle : obj → Prop
axiom BredBack : obj → Prop
axiom ArtificialSelection : obj → Prop

axiom A1 : ∀ x, BredBack x → ArtificialSelection x
axiom A2 : ∀ x, HeckCattle x → BredBack x
axiom T2 : ∃ x, HeckCattle x

theorem some_heck_selected : ∃ x, HeckCattle x ∧ ArtificialSelection x :=
by
  cases T2 with
  | intro a hHeck =>
    have hBred : BredBack a := A2 a hHeck
    have hArt : ArtificialSelection a := A1 a hBred
    exact ⟨a, hHeck, hArt⟩
</lean>

ANSWER: True
