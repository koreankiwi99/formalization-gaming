The context states that all squares are four-sided and all four-sided things are shapes. The statement to evaluate is that all squares are shapes. This can be proven as follows:  
- If something is a square, then it must be four-sided (from the first premise).  
- If it is four-sided, then it must be a shape (from the second premise).  
- Therefore, if something is a square, it is a shape.  

This is a straightforward transitive implication between properties. The Lean proof uses the axioms and applies them sequentially to demonstrate the conclusion.  

<lean>
axiom obj : Type
axiom Square : obj → Prop
axiom FourSided : obj → Prop
axiom Shape : obj → Prop
axiom T1 : ∀ x, Square x → FourSided x
axiom T2 : ∀ x, FourSided x → Shape x
theorem squares_are_shapes : ∀ x, Square x → Shape x :=
  fun x hSquare =>
    have hFourSided : FourSided x := T1 x hSquare
    T2 x hFourSided
</lean>  

ANSWER: True
