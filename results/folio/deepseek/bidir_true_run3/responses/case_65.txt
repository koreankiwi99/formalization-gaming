The context states that Yale University is a private Ivy League research university and that it moved to New Haven. This directly implies that there exists at least one private Ivy League research university (Yale) that moved to New Haven. The proof constructs Yale as a witness and applies the given properties.

<lean>
axiom obj : Type
axiom Yale : obj
axiom Private : obj → Prop
axiom IvyLeague : obj → Prop
axiom ResearchUniversity : obj → Prop
axiom MovedToNewHaven : obj → Prop

axiom Yale_private : Private Yale
axiom Yale_ivy : IvyLeague Yale
axiom Yale_research : ResearchUniversity Yale
axiom Yale_moved : MovedToNewHaven Yale

theorem exists_private_ivy_moved : ∃ x : obj, Private x ∧ IvyLeague x ∧ ResearchUniversity x ∧ MovedToNewHaven x :=
  by
    exists Yale
    exact ⟨Yale_private, Yale_ivy, Yale_research, Yale_moved⟩
</lean>

ANSWER: True
