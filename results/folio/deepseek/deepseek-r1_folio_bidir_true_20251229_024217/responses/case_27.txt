The Lean 4 translation captures the logical structure of the problem. The proof for `Alien Marvin` succeeds in the case where Marvin is from both Earth and Mars (via `from_mars_alien`), but fails in the case where Marvin is from neither (despite being extraterrestrial, we lack a premise that extraterrestrial implies alien). Thus, the statement "Marvin is an alien" is not provable from the given premises.

<lean>
axiom obj : Type
axiom Marvin : obj
axiom Alien : obj → Prop
axiom Extraterrestrial : obj → Prop
axiom Human : obj → Prop
axiom HighlyIntelligent : obj → Prop
axiom FromMars : obj → Prop
axiom FromEarth : obj → Prop

axiom all_aliens_are_extraterrestrials : ∀ x, Alien x → Extraterrestrial x
axiom from_mars_alien : ∀ x, FromMars x → Alien x
axiom no_extraterrestrial_human : ∀ x, Extraterrestrial x → ¬ Human x
axiom earth_highly_intelligent_human : ∀ x, (HighlyIntelligent x ∧ FromEarth x) → Human x
axiom marvin_highly_intelligent : HighlyIntelligent Marvin
axiom marvin_origin : (FromEarth Marvin ∧ FromMars Marvin) ∨ (¬ FromEarth Marvin ∧ ¬ FromMars Marvin)
axiom not_from_earth_extraterrestrial : ¬ FromEarth Marvin → Extraterrestrial Marvin

-- Attempt to prove Marvin is an alien
theorem marvin_alien : Alien Marvin := by
  cases marvin_origin
  · case inl h => 
      exact from_mars_alien Marvin h.right
  · case inr h => 
      have h_ext : Extraterrestrial Marvin := not_from_earth_extraterrestrial h.left
      -- Cannot derive Alien Marvin here: extraterrestrial does not imply alien
