-- Premises:
-- 1. All employees who schedule a meeting with their customers will go to the company building today.
axiom P1 : ∀ x : obj, Employee x → SchedulesMeetingWithCustomers x → GoToCompanyBuildingToday x

-- 2. Everyone who has lunch in the company building schedules meetings with their customers.
axiom P2 : ∀ x : obj, HasLunchInCompanyBuilding x → SchedulesMeetingWithCustomers x

-- 3. Employees have lunch either in the company building or at home.
axiom P3 : ∀ x : obj, Employee x → (HasLunchInCompanyBuilding x ∨ HasLunchAtHome x)

-- 4. If an employee has lunch at home, they are working remotely from home.
axiom P4 : ∀ x : obj, Employee x → HasLunchAtHome x → WorkingRemotelyFromHome x

-- 5. All employees who are in other countries work remotely from home.
axiom P5 : ∀ x : obj, Employee x → InOtherCountries x → WorkingRemotelyFromHome x

-- 6. No managers work remotely from home.
axiom P6 : ∀ x : obj, Manager x → ¬ WorkingRemotelyFromHome x

-- 7. James will appear in the company today if and only if he is a manager.
axiom P7 : AppearInCompanyToday James ↔ Manager James

-- Conclusion:
-- If James is either a manager or in other countries, then James does not either has lunch at home or work remotely from home.
-- If James either has lunch at home or works remotely from home, then he is neither a manager nor does he work in other countries.
theorem goal :
  ((Manager James ∨ InOtherCountries James) → ¬ (HasLunchAtHome James ∨ WorkingRemotelyFromHome James))
  ∧
  ((HasLunchAtHome James ∨ WorkingRemotelyFromHome James) → (¬ Manager James ∧ ¬ InOtherCountries James)) := sorry
