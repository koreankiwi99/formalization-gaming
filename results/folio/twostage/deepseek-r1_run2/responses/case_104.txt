=== REASONING ===
We are given the following axioms:
  R1: For any person p, if p is a GRE applicant and proves economic hardship, then ETS provides financial aid to p.
  R2: For any person p, if p lives in a single-parent family OR has few resources, then p can prove economic hardship.
  T1: Tom lives in a single-parent family.
  T2: Tom has few resources.
  T3: Tom is a GRE applicant.

We wish to prove: FinancialAidFromETS Tom.

Proof outline:
1. From T1 and T2, we know that Tom lives in a single-parent family and has few resources.
2. By R2, we need to show that Tom lives in a single-parent family OR has few resources. Since we have both, we can use either. We can use the left disjunct (T1) or the right disjunct (T2). Actually, we can use `Or.inl T1` to show the left disjunct, but note that R2 requires a disjunction. However, having both allows us to form the disjunction in multiple ways. We can use `LiveInSingleParentFamily Tom ∨ HaveFewResources Tom` by either proving the left (with T1) or the right (with T2). But actually, we can use both: we can use the fact that `LiveInSingleParentFamily Tom` is true to form `Or.inl T1` (which is a proof of `LiveInSingleParentFamily Tom ∨ HaveFewResources Tom`).

Alternatively, we can note that since we have T1, we have at least one of the two (in fact, we have both, but we only need one). So we can do:

  have H1 : LiveInSingleParentFamily Tom ∨ HaveFewResources Tom := Or.inl T1

But note: we can also use `Or.inr T2`. Either would work because R2 only requires one of them. However, we have both so we can choose either.

But actually, we don't need to choose: we can use the fact that we have T1 to form the disjunction. Let's use:

  have H_disj : LiveInSingleParentFamily Tom ∨ HaveFewResources Tom := Or.inl T1

Then by R2 applied to Tom and H_disj, we get:

  have H_hardship : ProveEconomicHardship Tom := R2 Tom H_disj

Then by R1 applied to Tom (with T3 and H_hardship), we get:

  FinancialAidFromETS Tom := R1 Tom T3 H_hardship

So the entire proof term can be written as:

  R1 Tom T3 (R2 Tom (Or.inl T1))

Alternatively, we could have used `Or.inr T2`:

  R1 Tom T3 (R2 Tom (Or.inr T2))

Both are valid.

Let's write the complete code accordingly.

Note: We must not change the axioms or the theorem statement.

We'll choose one of the disjunction proofs arbitrarily (say, using T1 via Or.inl).
