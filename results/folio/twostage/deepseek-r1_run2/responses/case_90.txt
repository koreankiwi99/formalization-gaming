=== REASONING ===
We are given the following axioms:
- `NLP_task` : Type
- `LG` and `LU` are properties of NLP tasks (predicates)
- `OutputTextSequence` is a predicate indicating that the task outputs a text sequence
- `MT` is a specific NLP task (constant)
- `categorization` : every task is either LG or LU
- `output_implies_LG` : if a task outputs a text sequence, then it is LG
- `mt_output` : MT outputs a text sequence

We wish to prove `LG MT`.

Proof:
Since we have `mt_output : OutputTextSequence MT` and `output_implies_LG` says that for any task `t`, if `OutputTextSequence t` then `LG t`, we can apply `output_implies_LG` to `MT` and `mt_output` to get `LG MT`.

Therefore, the proof term is: `output_implies_LG MT mt_output`

We do not need to use the `categorization` axiom because we have a direct implication from `OutputTextSequence MT` to `LG MT`.

Now, we write the complete code with the proof.

axiom OutputTextSequence : NLP_task → Prop

axiom output_implies_LG : ∀ t : NLP_task, OutputTextSequence t → LG t
