<lean>
axiom obj : Type
axiom Hamden : obj

axiom Listed : obj → Prop
axiom ManyNeg : obj → Prop
axiom RatingHigh : obj → Prop
axiom ProvidesTakeOut : obj → Prop
axiom Popular : obj → Prop

-- If a restaurant is listed in Yelp’s recommendations, then it has not received many negative reviews.
axiom A1 : ∀ x : obj, Listed x → ¬ManyNeg x
-- All restaurants with a rating greater than four are listed in Yelp’s recommendations.
axiom A2 : ∀ x : obj, RatingHigh x → Listed x
-- Some restaurants that do not provide take-out service receive many negative reviews.
axiom A3 : ∃ x : obj, ¬ProvidesTakeOut x ∧ ManyNeg x
-- All restaurants that are popular among local residents have ratings greater than four.
axiom A4 : ∀ x : obj, Popular x → RatingHigh x
-- The Hamden Plaza Subway store has a rating greater than four, or it is popular among local residents.
axiom A5 : RatingHigh Hamden ∨ Popular Hamden

theorem hamden_rating : RatingHigh Hamden :=
by
  cases A5 with
  | inl h => exact h
  | inr hp => exact A4 Hamden hp

theorem hamden_listed : Listed Hamden := A2 Hamden hamden_rating
theorem hamden_not_manyneg : ¬ManyNeg Hamden := A1 Hamden hamden_listed

-- We are asked to prove the negation of the statement:
-- Statement: ¬ProvidesTakeOut Hamden
-- Negation to prove: ProvidesTakeOut Hamden
-- From the given axioms, there is no way to derive ProvidesTakeOut Hamden.
-- A3 only provides existence of some restaurant without take-out and with many negative reviews,
-- which need not be Hamden (indeed, we derived ¬ManyNeg Hamden).
-- Therefore, we cannot prove ProvidesTakeOut Hamden from the premises.
</lean>
