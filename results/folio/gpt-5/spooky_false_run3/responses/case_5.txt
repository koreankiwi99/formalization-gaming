-- Predicates
axiom Employee : Person → Prop
axiom Manager : Person → Prop
axiom SchedulesMeetingWithCustomers : Person → Prop
axiom AppearsInCompanyToday : Person → Prop
axiom HasLunchInCompanyBuilding : Person → Prop
axiom HasLunchAtHome : Person → Prop
axiom WorksRemotelyFromHome : Person → Prop
axiom InOtherCountries : Person → Prop

-- Premises from the textual context
-- 1. All employees who schedule a meeting with their customers will go to the company building today.
axiom P1 : ∀ x : Person, Employee x ∧ SchedulesMeetingWithCustomers x → AppearsInCompanyToday x

-- 2. Everyone who has lunch in the company building schedules meetings with their customers.
axiom P2 : ∀ x : Person, HasLunchInCompanyBuilding x → SchedulesMeetingWithCustomers x

-- 3. Employees have lunch either in the company building or at home.
axiom P3 : ∀ x : Person, Employee x → HasLunchInCompanyBuilding x ∨ HasLunchAtHome x

-- 4. If an employee has lunch at home, they are working remotely from home.
axiom P4 : ∀ x : Person, Employee x ∧ HasLunchAtHome x → WorksRemotelyFromHome x

-- 5. All employees who are in other countries work remotely from home.
axiom P5 : ∀ x : Person, Employee x ∧ InOtherCountries x → WorksRemotelyFromHome x

-- 6. No managers work remotely from home.
axiom P6 : ∀ x : Person, Manager x → ¬WorksRemotelyFromHome x

-- 7. James will appear in the company today iff he is a manager.
axiom P7 : AppearsInCompanyToday James ↔ Manager James

-- Statement to evaluate (as a proposition about James)
-- If James is either a manager or in other countries, then James does not either have lunch at home or work remotely from home.
-- If James either has lunch at home or works remotely from home, then he is neither a manager nor does he work in other countries.
def Claim : Prop :=
  ((Manager James ∨ InOtherCountries James) → ¬(HasLunchAtHome James ∨ WorksRemotelyFromHome James))
  ∧ ((HasLunchAtHome James ∨ WorksRemotelyFromHome James) → (¬Manager James ∧ ¬InOtherCountries James))
